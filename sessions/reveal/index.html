Terminal Navigation

  • Open your terminal
  • Type:
    cd./desktop/repositories/username.github.io

Online >> Terminal

Copy this text and paste it into your terminal.

What we're typing

  • echo "# first" >> README.md
  • git init
  • git add README.md
  • git commit -m "first commit"
  • git remote add origin username.github.io
  • git push -u origin master

Setting your commit email as git email!

  • git config --global user.email "MY-NAME@example.com"

Adding in an actual webpage

  • Save a Sublime Text HTML file into you're folder and call it index.html

  • Add the tags and some stuff in the body

Pushing the changes

  • In your terminal make sure you're in the right folder: username.github.io
  • type: git status
  • type: git add .
  • type: git commit -m "adding index.html file"
  • type:git push

Making your repo a publishing source

  • go back to the github webpage
  • Scroll to "Github Pages"
  • For source: under dropdown menu select "master"
