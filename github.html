  1. Create a Github Account:

    To store our code and to have it be displayed as a website on the internet, we use a tool called Github.

    Github is a collaboration tool for software engineers. It's kind of like the coding version of Google Docs which programmers can use to collaboratively to edit code. Here is a video that describes how software engineers use Github.

    You can think of github kind of like icloud. You're going to have a folder on your computer that connects to Github the website so that you can "push" stuff to the internet. This will allow you to make changes to your code on your computer (i.e. locally) and then "push" it online so the new code shows up on your website on the internet.

    To set up a Github account go to github.com and create an account. For your username, please use your Dalton username (e.g. C24mj).

    Create a password and write it down in the notes app on your computer like this:

    github username: c22mj
    github password: whateverYourPasswordIs

    The reason we're doing this is so that when you forget your password, we can search your notes for "github password" and look it up. If you forget your password you can get locked out of your account.

  2. Create A Repositories Folder:

    Create a folder on your desktop called "repositories". A repository is a place where you store stuff. In this folder, we're going to store other folders. Each of them will store the code for the websites you make.

    To make a "repositories" folder on your desktop, go to your the "file" option in your menu bar, select "new folder". When it shows up on your desktop as "untitled" click on the world untitled once and it'll allow you to rename it, "repositories"

    Within the "repositories" folder, create another folder that is named yourUsername.github.io (i.e. If you're usename is c24mj it'll look something like c24mj.github.io). This is the folder that we are going to connect to github.
